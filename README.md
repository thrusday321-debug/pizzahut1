This project contains sql scripts to analyze Pizzahut sales data. It includes data ordered, order date, the quantity and insights generation for buisness decision-making.
DATASET STRUCTURE
orders : contain customer order details
order_details : contains pizza price
pizza_types : contain the name of the pizza and category as well.
# Key queries included
1. Total revenue analysis
2. Most ordered pizza types
3. Hourly/Monthly trend Analysis
4. Category wise revenue contribution.
